# CancelaIA

Gestor inteligente de suscripciones con escaneo de emails y cancelacion con un clic.

## Stack

- Next.js 14 + TypeScript + Tailwind CSS
- - Supabase (base de datos)
  - - Stripe (pagos y suscripciones)
    - - Google OAuth 2.0 + NextAuth.js
      - - Gmail API + Claude API
        - - Railway (deploy)
